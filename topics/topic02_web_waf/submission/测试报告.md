# IGA-Guard 3.0 测试报告

> **作品名称**：IGA-Guard 3.0 — 面向混淆逃逸攻击的可解释自演化 Web 安全防御系统  
> **赛题**：Web 攻击载荷混淆逃逸检测  
> **测试日期**：2026-07-04  
> **测试环境**：Windows 10 · Python 3.13.2 · CPU 推理

---

## 摘要

本报告对 IGA-Guard 3.0 进行功能测试、性能测试、离线评测与消融实验验证。全量诚实评测（n=19,411）显示混淆子集 Recall **99.95%**（达标 >99.5%），混淆 Precision **100%**，Normal FPR **5.63%**，单请求延迟 P50 **2.92 ms**（达标 <5 ms），WebSpotter 定位 IoU 提升 **+37.9%**（达标 ≥+22%）。单元测试 **32/32 通过**。

## 六、消融与对比实验

### 6.1 多模态消融（全量 n=19,411）

**结果文件**：`results/v2_compare_multimodal_full.json`

| 配置 | 混淆 Recall | Normal FPR |
|------|-------------|------------|
| 关闭多模态 | 92.17% | 3.10% |
| **开启多模态（完整）** | **99.95%** | 5.63% |
| Δ（混淆 Recall） | **+7.78 pp** | — |

条件融合策略：混淆样本压低多模态权重（4%），避免视觉轨误报；非混淆样本加强多模态压 FP。

## 七、测试结论

### 7.1 达标项

| 指标 | 目标 | 实测 | 结论 |
|------|------|------|------|
| 混淆子集 Recall | > 99.5% | **99.95%** | ✅ 达标 |
| 混淆 Precision | — | **100%** | ✅ 优秀 |
| 延迟 P50 | < 5 ms | **2.92 ms** | ✅ 达标 |
| 定位 IoU 提升 | ≥ +22% | **+37.9%** | ✅ 达标 |
| 单元测试 | 全通过 | **32/32** | ✅ 达标 |
| 虚拟补丁 | 有效拦截 | **90%** | ✅ 良好 |

### 7.2 待优化项

| 指标 | 现状 | 改进方向 |
|------|------|----------|
| Normal FPR | 5.63% | CSIC 表单护栏、url_encode 规则收紧，目标 <3% |
| P99 延迟 | 27.38 ms | 模型预热、INT8 量化 |
| 剩余漏检 | 5 条 | concat/hex 边界变种专项规则 |

### 7.3 总体评价

IGA-Guard 3.0 在赛题核心指标「混淆子集检出率」上达到 **99.95%**，优于 ModSecurity CRS（文献 TPR ~68%）与传统 ML 基线（70~80%）。系统具备完整的 API、可视化大屏、混淆生成器与对抗演化闭环，**32 项单元测试全部通过**，可作为可运行的 Web 攻击混淆逃逸检测演示系统提交。
