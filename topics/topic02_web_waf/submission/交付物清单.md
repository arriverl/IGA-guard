## 六、导出 PDF 建议

1. 用 VS Code / Typora / Pandoc 将 `.md` 导出为 PDF

3. 原创性声明：浏览器打开 `原创性声明.svg` → 打印 → 手写签名盖章 → 扫描为 PDF

```powershell

pandoc submission/作品报告.md -o submission/作品报告.pdf
pandoc submission/运行说明.md -o submission/运行说明.pdf
pandoc submission/测试报告.md -o submission/测试报告.pdf
```
