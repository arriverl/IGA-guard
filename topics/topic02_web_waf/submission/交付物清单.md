## 三、系统能力交付（Dynamic Guard 终稿）

### 3.1 在线检测链路

```
HTTP/Multipart → 协议展开 → 多层解混淆(decode_chain) → 四模态融合
  → 混淆 Boost / FP 护栏 → WebSpotter 定位 → 规则导出 / 虚拟补丁
```

### 3.2 已交付核心模块

| 模块 | 路径 | 说明 |
|------|------|------|
| 主流水线 | `src/iga_guard/pipeline.py` | `IgaGuardEngine` 统一入口 |
| 多层解混淆 | `src/iga_guard/normalizer/` | URL/atob/实体/null 间隔/JPG 头剥离 |
| 四模态融合 | `src/iga_guard/detector/dual_track.py` | RF + TinyBERT + 多模态 + DLinear |
| XXE 精准识别 | `src/iga_guard/obfuscation_signals.py` | hex 实体、参数实体、本地 DTD、SVG 上传 |
| Multipart 上传 | `src/iga_guard/collector/` | WAFFLED 协议轨；file 字段优先检测 |
| Miss→Rule 闭环 | `src/iga_guard/evolution/miss_rule_pipeline.py` | 漏检聚类 → 动态 rescue 规则 |
| 持续学习缓存 | `src/iga_guard/evolution/continual_cache.py` | Tip-Adapter KV 库 |
| LLM 红队 | `scripts/run_llm_redteam.py` | E9 对抗实验 |
| RAG 知识增强 | `src/iga_guard/rag/` + `scripts/build_rag_index.py` | 手法库检索 |
| Web 可视化 | `frontend/static/p1~p6_*.html` | 监控/详情/定位/风险/演化/规则 |
| VPS 部署 | `deploy/`（仓库根目录） | inline 代理 + gunicorn + systemd |
| 全自动门禁 | `scripts/run_auto_verify.py` | pytest + E1~E9 一键验收 |

### 3.3 支持的混淆类型（动态识别）

| 类型 | 示例 | decode_chain / 机制 |
|------|------|---------------------|
| 多重 URL 编码 | `%2527 union select` | `url_decode` × N |
| Null-byte 间隔 | `%00%3c%00%3f...` | `null_interleave_strip` |
| atob / Base64 | `eval(atob('...'))` | `atob_decode` |
| XML 十六进制实体 | `&#x53;&#x59;...`（SYSTEM） | `xml_entity_expand` |
| 参数实体拆分 | `SYS` + `TEM` → SYSTEM | `xxe_rescue_label` |
| JPG 伪装上传 | `ÿØÿÛ` + SVG/XXE | `upload_magic_strip` |
| HPP / JSON 嵌套 | WAFFLED 协议轨 | `protocol_normalize.py` |

## 六、演示验收命令（评委可复现）

### 6.1 环境与服务

```powershell
cd topics\topic02_web_waf
pip install -r requirements.txt
$env:PYTHONPATH="src"
python run.py
# 浏览器: http://127.0.0.1:5000/
```

### 6.2 三类代表性攻击（建议答辩顺序）

| 场景 | 命令 / 操作 | 预期 |
|------|-------------|------|
| 双重 URL SQLi | `python scripts/detect.py --url "http://x/?q=1%2527%20union%20select" --json` | SQLi + decode_chain |
| Null-byte XXE | `python scripts/_test_xxe_null.py`（或 P2 粘贴 URL） | XXE + null_interleave_strip |
| Multipart SVG 上传 | `python scripts/test_multipart_xxe_api.py` | XXE + upload_magic_strip |

> Windows 勿在 CMD 直接 curl 含 `&`/`%` 的 multipart JSON；用上述 Python 脚本。

### 6.3 一键全量验收

```powershell
python scripts/run_auto_verify.py
python -m pytest tests/ -q
```
